# 📊 Retail Sales Performance Dashboard (2019–2020)

## 📌 Project Overview
This Power BI dashboard analyzes retail sales performance across 2019–2020.  
The objective was to identify revenue trends, profitability drivers, and category level performance insights.


## 🔹 Key KPIs
- Total Sales
- Prior Year Sales
- Growth %
- Sales YTD
- Profit Margin %


## 🔹 Key Insights
- Sales increased significantly in Q4 (Nov–Dec).
- Office Supplies contributes the highest revenue share.
- Technology generates the highest overall profit.
- Copiers lead profitability within the Technology segment.
- Furniture underperforms in margin efficiency.


## 🛠 Tools & Techniques Used
- Power BI
- Star Schema Data Modeling
- DAX Measures
- Time Intelligence (DATEADD, YoY Growth)
- Drill-Down Analysis
- Dynamic Filtering
